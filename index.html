WatchNext

This app allows users to search movies and read a short description about them. If they are interested, they can choose to add the movie to a list of movies to watch next or watch it right away via multiple sources. Go to the repo

Weather Dashboard

This app allows the user to find the current weather and a 5-day forcast of most cities. The search history is saved to local storage so the user can just click on previously searched cities. Go to the repo

Work Day Scheduler

This app allows users to schedule tasks for the current work day. The tasks are saved to local storage so the user can close the browser and reopen it to see the tasks. The work hours can be inserted or deleted to accomodate for earlier or later beginning and end times. Go to the repo
